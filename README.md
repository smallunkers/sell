# sell

> sell app

#####完成了一个外卖的spa 应用
用了vue 2.0 + vue-router + vue-resource + eslint + webpack
主要的组件包括 header 头组件，food 组件，goods 组件，ratings 组件，seller 组件，shopcart 购物车组件。
另外抽象复用的组件 cartcontrol 加入去除按钮组件， star 评分组件
另外引入better-scroll插件来控制滑动。
